# openclaw-cli-mcp

**Built with:** Python

## Getting Started

Clone the repository:

```bash
git clone https://github.com/suhteevah/openclaw-cli-mcp.git
cd openclaw-cli-mcp
```

## Installation

```bash
pip install -r requirements.txt
python main.py
```
